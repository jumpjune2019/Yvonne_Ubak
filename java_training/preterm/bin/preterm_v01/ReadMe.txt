This first version setups up the basic scaffolding and demonstrates inputting data from a CSV file.

SCAFFOLDING
start()
*enter the application
*ask to login or register or quit

getInput()
*receive client list --> from client class
*receive room matrix

assign()
*assign clients to a seat in a room

display()
*show clients list
*show room matrix - seat availability

logout()
*logout user
*return to start()

end()
*System.exit(0);
